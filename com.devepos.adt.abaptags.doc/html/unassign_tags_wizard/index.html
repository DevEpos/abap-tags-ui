Remove Tags from Objects Wizard

This wizard can be used to add Tags to Repository Objects. If it is called from the main Eclipse toolbar the initial page will be the Tag Selection Page.
However, if the wizard is called from the context menu of the Tagged Object Tree or the Object Tags View the initial page will be skipped and you can directly start on the Tagged Object Selection Page.

For this feature you have to install at least Version v2.2 of the ADT Backend abap-tags-backend

Pages of the Wizard

  1. Select ABAP Tags
  2. Select Tagged Objects
