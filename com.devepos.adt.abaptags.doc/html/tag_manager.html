Tag Manager

The Tag Manager view manages all available user, shared and global tags for a given ABAP project.
You can perform the following actions in the view:

  • create, edit or delete user and/or global tags
  • convert user tags into global tags
  • share user tags
  • trigger tagged object search from selected tags

Sharing User Tags

You can only share normal user tags and the root node of hierarchical user tags. Upon sharing the root all sub tags are also shared

Sometimes you may need to share a specific set of objects with only a few other users. In this case it may be better to create a user tag and share it with others.
The sharing process is started via the context menu action Share Tag... of a user tag.

After that you can select those users with who you want to the share the selected tag.

Deleting Tags

Starting with the Backend Version 2.0, Tags can no longer be deleted if there are still objects assigned to it or any child Tags of the selected Tag.
This should be viewed as an additional safe guard against accidental deletion of a Tag and its assigned Objects.

Upon triggering the deletion of one or several Tags you will see a wiard dialog which will give you an overview of the selected Tags, and more importantly it will show you if a selected Tag can be safely deleted or not
