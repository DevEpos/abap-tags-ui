Object Tags

The Object Tags View allows the viewing/changing of the current Tags of a given ADT object.

You can pin this view to a side of eclipse and activate the Link with Editor action in the View's toolbar.
That way, the view will automatically show the ADT object of the currently active editor.

Further possible actions in this View are:

  • Delete assigned tags
  • Choose any Repository Object for Tagging that is accessible by ADT

Tagged Components

When a global class contains local classes/interface that also have Tags assigned, the local objects are displayed in separate folder called Tagged Components.

Tags of components can be deleted via the context menu, just as a Tag of other repository objects.
