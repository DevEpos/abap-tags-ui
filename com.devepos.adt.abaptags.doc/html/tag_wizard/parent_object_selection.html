Parent Object Selection

On this page you choose the parent ADT objects for the selected hierarchical tags. You can do this either in single or multiple selection mode .

To select a parent object, you simple click into the table cell where it says Click to assign parent object. After that the following dialog will open, where you can search for the parent ADT object which you want to use.

If no parent object exists yet, for the parent tag, you cannot finish the wizard. A hierarchical tag without a parent object assignment is not possible.
