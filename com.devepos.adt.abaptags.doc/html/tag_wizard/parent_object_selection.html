Parent Object Selection

On this page you choose the parent Repository objects for the selected hierarchical tags. You can do this either in single or multiple selection mode.

To select a parent object, you simple click into the table cell where it says Click to assign parent object. After that the following dialog will open, where you can search for the parent Repository object which you want to use.
The filter field of the dialog supports filtering with wildcards ( * ). Additionally you can sort the found Objects via the column headers of the table.

The dialog lists all available tagged objects in a given hierarchy from the currently chosen tag to the root. This gives you the option to skip one or several levels during assignment
You can also decide to leave the parent empty. If you already assigned a parent object you can remove it via the context menu and finish the wizard.
